# 🌍 ExploreWorld - Travel Website

A modern and responsive travel website built using **HTML and CSS**.
This project showcases a complete front-end layout including navigation, video background, destination cards, packages, testimonials, and a booking form.

---



## 📌 Features

* 🎥 Hero section with background video
* 🧭 Sticky navigation bar
* 🌐 Smooth scrolling navigation
* 🗺️ Destinations section with cards
* 💼 Travel packages section
* 💬 Testimonials section
* 📝 Booking form UI
* 📱 Fully responsive design
* 🎨 Clean and modern UI

---

## 🛠️ Tech Stack

* **HTML5** → Structure of the website
* **CSS3** → Styling and layout
* **Flexbox** → Navbar alignment
* **CSS Grid** → Card layouts
* **Media Queries** → Responsive design
* **Font Awesome** → Icons
* **Google Fonts** → Typography
